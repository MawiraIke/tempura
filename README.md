## Quickstart

Add the necessary dependency to your project:

```clojure
[com.taoensso/tempura "1.0.0-SNAPSHOT"]
```

And setup your namespace imports:

```clojure
(def my-clj-or-cljs-ns
  (:require [taoensso.tempura :as tempura :refer [tr]]))
```
